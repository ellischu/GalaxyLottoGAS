- **核心限制**：GAS 單次執行 6 分鐘限制 (360 秒)，需強制採用**中斷續傳**與**批次 API 呼叫**設計。
- **前端界面**：jQuery + Bootstrap + Swal2 (SweetAlert2) 高感度回饋界面。
- **響應式適配**：
  - 手機端：1080 \* 2340 縱向高解析度優化。
  - 平板端：1280 \* 800 視差布局。
  - 桌面端：寬螢幕多模組並排顯示。
